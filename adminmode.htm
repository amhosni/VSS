To create an unattended Visual Source Safe Setup

 

1.       On this page:

a)      Read the end-user license agreement (EULA).

b)      Accept the conditions of the EULA.

c)       Enter your name.

d)      Click Continue.

 

2.       On the Options page, choose the features you would like to install.

 

3.       Save the generated install configuration to an .ini file and then deploy to remote computers.

For more information, see the Administrator Readme, adminreadme.htm, located in the root folder of the installation source.
