Visual SourceSafe

 

Visual SourceSafe is the ideal version control system for individuals and teams using Microsoft Visual Studio®. The costs and overhead associated with version control often outweigh the benefits. Visual SourceSafe provides a low-cost, project-oriented version management system, and enables software developers to work with the confidence that their projects and files will be protected. It is simple enough to use right out of the box, and it is integrated with the development environment developers already work in.

 

Visual SourceSafe features and benefits:

 

Begin working in minutes

Ø Installation, configuration, and user setup take minutes, all with effortless administration.

Ø Use the stand-alone SourceSafe Explorer, the command line, or even features integrated within the Visual Studio integrated development environment (IDE) and many third-party tools.

 

Promote code reuse among projects

Ø Simplify file sharing across several projects with share and link capabilities.

Ø As changes are made to one share, the changes automatically propagate to other shares, ensuring that common files are always up-to-date in every project.

 

Improve software quality for your projects

Ø Protect documents among team members. Check in and check out file locking securely protects files from accidental overwrite by preventing more than one user from modifying the same file at the same time.

Ø Access project versions easily. Labels provide snapshots of a project for quick retrieval anytime during the software life cycle.

Ø View differences across versions. Difference reporting provides quick, visually enhanced access to changes across different versions of the same file.

 

Maintain an audit trail for every file

Ø View the changes—when they were made and who made them—and even print reports.
